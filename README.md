# 🤖 AI Chatbot

This project is a simple AI chatbot built using Python and Flask.  
It uses NLP techniques to understand user queries and respond intelligently.

## 🚀 Features
- Natural Language Processing using NLTK
- JSON-based intents
- Interactive chat interface
- Lightweight and beginner-friendly

## 🛠️ Technologies Used
- Python
- Flask
- HTML, CSS, JavaScript
- NLTK

## 📂 Project Structure
- app.py → Main backend file
- intents.json → Chatbot training data
- templates/ → Frontend UI

## ▶️ How to Run
1. Install Python
2. Install required libraries:
   pip install flask nltk
3. Run:
   python app.py
